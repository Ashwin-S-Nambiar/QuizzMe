## üöÄ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/quizzme.git
   cd quizzme
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   **The application will be accessible at `http://localhost:5173`.**
